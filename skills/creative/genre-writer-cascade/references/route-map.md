# Genre route map

Use explicit `--genre` values whenever the user has named the primary promise.
Automatic matching exists for ordinary intake, but a tie intentionally stops.

| Canonical genre | Accepted aliases and strong signals | Profile |
|---|---|---|
| romance | romantic comedy, romantasy when relationship arc is primary | heartline |
| fantasy | epic fantasy, urban fantasy, sword and sorcery, magic, mythology | spellbound |
| science-fiction | sci-fi, space opera, cyberpunk, speculative technology | orbit |
| horror | gothic horror, cosmic horror, supernatural horror, dread | nocturne |
| mystery-thriller | mystery, detective, crime, suspense, thriller, whodunit | casefile |

For romantasy, gothic romance, science-fantasy, and horror-thriller, state the
primary reader promise explicitly. Keep the other label as secondary metadata.
