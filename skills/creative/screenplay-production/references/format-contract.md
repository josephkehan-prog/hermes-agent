# Script format contract

## Screen

- Slugline: `INT./EXT. LOCATION - TIME`
- Action: present tense, visible or audible, short paragraphs.
- Character cue: stable canonical name.
- Dialogue: playable speech; parentheticals only when meaning would otherwise fail.
- Transition: use sparingly when editorial intent matters.

## Stage

- Name act/scene and physical setting.
- Separate dialogue from executable stage direction.
- Track entrances, exits, costume, props, and scene-change requirements.

## Scripted audio

- Identify speaker, delivery intent, music/SFX cue, and silence.
- Keep spoken copy distinct from production direction.
- Carry stable cue IDs into narration, sound design, and caption files.

Never infer industry-format compliance from appearance alone. Record format,
page count, estimated runtime, and the tool used to render or inspect the file.
