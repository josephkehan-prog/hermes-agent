# Voice ledger

Record one row per stable voice assignment:

| Field | Requirement |
|---|---|
| voice_id | Exact Kokoro voice name |
| language | Kokoro language code |
| role | Narrator or fictional character |
| speed | Exact numeric value |
| pronunciation | Approved exceptional pronunciations |
| source_scope | Chapters/scenes using this voice |
| consent | `synthetic-stock`, or documented authority for supplied voice material |

Use one narrator voice unless distinct voices materially aid comprehension.
Never represent a stock synthetic voice as a real performer.
